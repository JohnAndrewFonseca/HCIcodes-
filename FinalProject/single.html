Keyboard Layout

The term "keyboard size" or layout describes the actual size of the keyboard, which is based on the number of keys on it. With some variance in how layouts compress them closer together and vice versa, a keyboard's general size increases with the number of physical keys it contains.

Sizes are sometimes expressed as percentages, such as 100%, TKL or Tenkeyless, 75%, 60%, 40%, and so on, with a standard full-size layout serving as the benchmark.

For languages using the Latin alphabet, there are three main key layouts: QWERTY, QWERTZ, and AZERTY. Regional variations in several countries have been created based on these. For example, the CSA (Canadian French) keyboard uses QWERTY.
